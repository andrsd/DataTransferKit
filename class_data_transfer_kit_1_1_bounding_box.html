Detailed Description

Axis-aligned Cartesian bounding box container.

All three dimensions are explictly represented in this bounding box, however, from an algorithmic standpoint, this can be treated as a one or two dimensional box as well by setting the unused dimension bounds to +/- Teuchos::ScalarTraits<double>::rmax().

double DataTransferKit::BoundingBox::volume ( const int  dim) const

Compute the volume of the bounding box given its dimension.

Parameters:
dimThe dimension of the box we want to compute the volume for. We need this because the box always stores all 3 dimensions. Lower dimension boxes are resolved with higher dimensions set to +/- Teuchos::ScalarTraits<double>::rmax(). For dim = 1, only the x dimension is used. For dim = 2, the x and y dimensions are used. For dim = 3, the x, y, and z dimensions are used.
Returns:
Return the volume of the box.

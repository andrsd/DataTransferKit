Detailed Description

template<typename GlobalOrdinal>
class DataTransferKit::KDTree< GlobalOrdinal >

A KDTree data structure for local mesh searching in the rendezvous decomposition.

A kD-tree provides n*log(n) construction time and space complexity with log(n) search time complexity.

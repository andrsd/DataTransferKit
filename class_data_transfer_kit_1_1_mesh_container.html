Detailed Description

template<typename GlobalOrdinal>
class DataTransferKit::MeshContainer< GlobalOrdinal >

A default mesh implementation.

The MeshContainer is a default implementation of MeshTraits for clients. Its data mimics the structure of MeshTraits with constructor arguments expected to have the data layout of a MeshTraits object. Each block of mesh must be in a separate mesh container.

Evaluate the field in the given elements at the given coordinates and return the evaluations in a Field.

Parameters:
elementsA vector of element handles in which to evaluate the field.
coordsA vector of blocked coordinates ( x0, x1, x2, ... , xN, y0, y1, y2, ... , yN, z0, z1, z2, ... , zN ) at which to evaluate the field. Coordinates ( xN, yN, zN ) should be evaluated in the Nth element in the elements vector.
Returns:
Return a Field containing the evaluated field values. This returned field is required to be of the same length as the elements input vector. For those coordinates that can't be evaluated in the given element, return 0 in their position. Field data dimensionality and ordering is specified by field traits.

Check whether two communicators own the same communication space.

Parameters:
comm_ACommunicator A.
comm_BCommunicator B.
Returns:
Return true if communicator A and B operate on the same group of processes. These processes do not have to have the same ranks in the communicators to be considered equivalent, they are only required to be the same physical process. The same result is produced independent of the ordering of A and B in the input parameters.
